Response class

This is just a copy of the HttpRequest object. I've extended it with the ability to add attributes to the request to support filter.

class Response implements HttpResponse {
 Response(HttpResponse this._httpResponse) {
   _mirror = reflect(this._httpResponse);

   this.close = () {
     return this._httpResponse.close();
   };
   this.write = (Object obj) {
     this._httpResponse.write(obj);
   };
 }

 dynamic noSuchMethod(Invocation invocation) {
   return _mirror.delegate(invocation);
 }

 CloseFunction close;
 WriteFunction write;

dynamic noSuchMethod(Invocation invocation) #

noSuchMethod is invoked when users invoke a non-existant method on an object. The name of the method and the arguments of the invocation are passed to noSuchMethod in an Invocation. If noSuchMethod returns a value, that value becomes the result of the original invocation.

The default behavior of noSuchMethod is to throw a noSuchMethodError.
